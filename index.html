Shortcut instellen

Maak eenmalig een shortcut aan in de Opdrachten-app zodat je vanuit élke app tekst kunt delen naar je bundel.

Stap 1 — Open de Opdrachten-app op je iPad

Stap 2 — Tik op + rechtsboven

Stap 3 — Tik op de naam bovenin → typ Opslaan in EPUB

Stap 4 — Tik op het ⚙️-icoon → zet Gebruik in deelmenu aan → Invoertypen: alleen Tekst

Stap 5 — Tik op Voeg actie toe en voeg de volgende acties toe in volgorde:

  1. Zoek Vraag om invoer → Vraag: "Titel van het artikel?" → Sla op als variabele Titel
  2. Zoek Vraag om invoer → Vraag: "Categorie?" → Standaard: "Algemeen" → Sla op als variabele Categorie
  3. Zoek Huidige datum → Voeg toe
  4. Zoek Datum opmaken → Opmaak: Aangepast → yyyy-MM-dd → Sla op als variabele Datum
  5. Zoek Datum opmaken (tweede keer) → Opmaak: Aangepast → HHmmss → Sla op als variabele Tijd
  6. Zoek Tekst → Voer in: 
    TITEL: [Titel]
DATUM: [Datum]
CATEGORIE: [Categorie]
---
[Opdracht-invoer]
    Sla op als variabele Inhoud
  7. Zoek Codeer → Codeer met Base64 → invoer: variabele Inhoud → Sla op als variabele Gecodeerd
  8. Zoek Haal inhoud van URL op → stel in:
    • URL: https://api.github.com/repos/FreyaGoogle/epub-bundel/contents/input/[Datum]_[Tijd].txt
    • Methode: PUT
    • Kopteksten: voeg toe → Sleutel: Authorization → Waarde: token jouw-token
    • Kopteksten: voeg toe → Sleutel: Accept → Waarde: application/vnd.github+json
    • Aanvraaginhoud: JSON → voeg velden toe:
      • message → Artikel: [Titel]
      • content → variabele Gecodeerd
      • branch → main
  9. Zoek Toon resultaat → Tekst: "Opgeslagen in bundel!"
